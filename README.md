# matplotlib

Приложение позволяет визуализировать трёхмерные данные о прибыли компаний в разные годы по срезам в виде различных типов диаграмм.

В файле **"GUI приложения.png"** пример предварительного заполнения формы для анализа данных и вывода результатов.

В файле **"data.xlsx"** содержатся трёхмерные данные о прибыли компаний для анализа. Измерения: регионы, компании, годы, то есть каждое значение - размер прибыли конкретной компании в конкретном регионе за конкретный год.

В файле **"metadata.xlsx"** содержатся метаданные для базы данных: названия регионов и компаний, а также годы, за которые получена прибыль.

В файле **"main.py"** весь Python-код программы:
- Для визуализации используется **matplotlib**, для графического интерфейса - pyqt5
- Используемые типы диаграмм: **график** (для временных рядов), **круговая** (для долевого вклада), **столбчатая** (для сравнения нескольких категорий); возможные типы диаграмм подбираются в зависимости от выбранного типа среза
- Для использования приложения необходимо заполнить форму: в блоке "Срез" выбрать **измерение**, по которому будет проведён срез, **значение среза** и **тип диаграммы** для отображения результатов; в блоке "Отображаемые элементы" выбрать **значения остальных измерений**, которые необходимо визуализировать
